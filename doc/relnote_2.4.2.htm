RTKLIB: Release Notes 2013/03/12

RTKLIB ver.2.4.2 Release Notes

New Features in ver.2.4.2

(1) Galileo and BeiDou are supported.
(2) Full RINEX 3 complient. Multiple-code in a frequency is supported.
(3) RTCM 3 MSM, SSR and ephemerides for All GNSS are supported.
(4) NovAtel OEM6, NVS NVC08 BINR and BINEX messages are supported.
(5) Google Earth View and Goolge Map View are added to RTKPLOT
(6) NORAD TLE (two-line element) data set is supported for satellite positions.
(7) Downloader and launcher APs are added (RTKGET and RTKLAUNCH).
(8) Data format conversion is supported in STRSVR and STR2STR.
(9) PPP-AR (ambiguity resolution) is supported. (experimental).
(10) Satellite visibility and multipath analysis function is added in RTKPLOT.
(19) Variable SNR mask and RAIM FDE (fault detection and exclution) is added.

Changes: ver.2.4.1 -> ver.2.4.2

(1) The following patches for ver.2.4.1 are included:
rtklib_2.4.1_p1, rtklib_2.4.1_p2, rtklib_2.4.1_p3, rtklib_2.4.1_p4, rtklib_2.4.1_p5, rtklib_2.4.1_p6, rtklib_2.4.1_p7,
(2) The following source codes are added for RTKLIB library:
src/ppp_ar.c, src/qzslex.c, src/rtcm2.c, src/rtcm3.c, src/rtcm3e.c, src/tle.c, src/rcv/binex.c, src/rcv/nvs.c
(3) The following APIs are added, deleted and modified:
(a) Added: satexclude(), testsnr(), setcodepri(), getcodepri(), obs2code(), code2obs(), bdt2time(), time2bdt(), gpst2bdt(), bdt2gpst(), time2gmst(), outrnxcnavh(), init_rnxctr(), free_rnxctr(), open_rnxctr(), input_rnxctr(), tle_read(), tle_name_read(), tle_pos(), setbitu(), setbits(), crc16(), input_binex(), input_nvs(), input_binexf(), input_nvsf(), gen_nvs(), gen_rtcm2(), gen_rtcm3(), pppos(), ppnx(), pppoutsolstat(), windupcorr(), pppamb(), strconvnew(), strconvfree(), dl_readurls(), dl_readstas(), dl_exec(), dl_test(), lexupdatecorr(), lexreadmsg(), lexoutmsg(), lexconvbin(), lexeph2pos(), lexioncorr()
(b) Deleted: setrnxcodepri()
(c) Modified: obs2code(), code2obs(), antmodel(), readrnx(), readrnxt(), readsp3(), eci2ecef(), sunmoonpos()
(4) The build environment for Windows APs is changed from Turbo C++ to C++ builder XE2 or XE3. Turbo C++ is no longer supported.
(5) RTKNAVI, RTKNAV_MKL
(a) Galileo and BeiDou are supported.
(b) RTCM 3 MSM and MSM draft are supported for inputs.
(c) RTCM 3 Galileo and QZSS ephemerides are supported for inputs.
(d) NovAtel OEM6, NVS NV08C BINR and BINEX are supported as inputs.
(e) Variable SNR mask, PPP options and RAIM FDE are supported.
(f) "Sync Solution" option is added.
(g) NMEA sentences GLGSA, GAGSA, GLCSV, GACSV are supported for outputs.
(h) TLE data are supported for skyplots
(i) Message dump functions are added for input stream monitor.
(j) Variable window size is supported.
(6) RTKPOST, RTKPOST_MKL
(a) Galileo and BeiDou are supported.
(b) RINEX 3.01 and 3.02 draft are supported for inputs.
(c) Variable SNR mask, PPP options and RAIM FDE are supported.
(d) OTL and pole tide corrections are supported.
(e) RTCM SSR corrections are supported for ephemeris and clock
(f) PPP-AR is supported (experimental)
(g) RINEX options are supported for inputs.
(7) RTKCONV
(a) Galileo and BeiDou are supported.
(b) Full RINEX 3 complient. Multi-code in a frequency is supported.
(c) RTCM 3 MSM and MSM draft are supported for inputs.
(d) RTCM 3 Galileo and QZSS ephemerides are supported for inputs.
(e) NovAtel OEM6, NVS NV08C BINR and BINEX are supported as inputs.
(f) RINEX 2.10, 2.12, 3.01, 3.02 are supported as output format.
(g) "Scan Obs Type" feature is added.
(h) Signal masks are added for manual signal selection.
(8) STRSVR
(a) Format conversion options are added.
(9) RTKPLOT
(a) Galileo and BeiDou are supported.
(b) Multi-code in a frequency is supported.
(c) Satellite visbility analysis is added.
(d) Multpath analysis functions are added.
(h) TLE data are supported for satellite positions
(i) Circle grids are supported.
(j) Google Map View and Google Earth View are added.
(10) SRCTBLBROWS
(a) Map view is added.
(11) RTKGET
Newly added.
(12) RTKLAUNCH
Newly added.
(12) RTKRCV
(a) The same as (a)(b)(c)(d)(e)(f)(g) of RTKNAVI.
(13) RNX2RTKP
(a) The same as (a)(b)(c)(d)(e)(f)(g) of RTKPOST
(14) CONVBIN
(a) The same as (a)(b)(c)(d)(e)(f) of RTKCONV.
(b) Option -scan, -oi, -ot, -ol are added.
(15) STR2STR
(a) Format conversion options are added.
(b) Option -msg, -opt, -sta are added.
(c) Option -p is modified.
(16) Utility
(a) Utility PREC2NAV is added.
(b) Utiliity RNX2RTCM is added.
